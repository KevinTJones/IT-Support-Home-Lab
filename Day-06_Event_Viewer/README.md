## Day 6 – Event Viewer Log Analysis

This lab focuses on using Windows Event Viewer to identify, filter, and review system warnings as part of routine troubleshooting and system health monitoring.

---

### Tasks Completed
- Opened Windows Event Viewer
- Reviewed Windows Logs → System
- Filtered logs to display Warning-level events
- Identified frequent system warnings and event sources
- Documented findings for troubleshooting and escalation reference

---

### Tools Used
- Windows 11
- Event Viewer

---

### Skills Demonstrated
- Log analysis
- System troubleshooting
- Identifying warning-level events
- IT documentation and evidence collection
